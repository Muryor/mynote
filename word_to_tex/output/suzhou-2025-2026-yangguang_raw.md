4．在$$\bigtriangleup ABC$$中，"$$\angle A$$为锐角"是"$$\left| \overrightarrow{AB} + \overrightarrow{AC} \right| > \left| \overrightarrow{BC} \right|$$"的（    ）

> A．充分必要条件 B．充分不必要条件
>
> C．必要不充分条件 D．既不充分也不必要条件

【答案】A

【难度】0.65

【知识点】充要条件的证明、向量减法的法则、平面向量数量积的定义及辨析

【分析】根据向量的运算及充分必要条件的定义即可判断；

【详解】对于充分性，若$$\angle A$$为锐角，则$$\overrightarrow{AB} \cdot \overrightarrow{AC} = \frac{1}{4}\left\lbrack \left( \overrightarrow{AB} + \overrightarrow{AC} \right)^{2} - \left( \overrightarrow{AB} - \overrightarrow{AC} \right)^{2} \right\rbrack > 0$$，

所以$$\left( \overrightarrow{AB} + \overrightarrow{AC} \right)^{2} > {\overrightarrow{BC}}^{2}$$，即$$\left| \overrightarrow{AB} + \overrightarrow{AC} \right| > \left| \overrightarrow{BC} \right|$$，故充分性正确；

对于必要性，若$$\left| \overrightarrow{AB} + \overrightarrow{AC} \right| > \left| \overrightarrow{BC} \right|$$，两边平方得$$\left( \overrightarrow{AB} + \overrightarrow{AC} \right)^{2} > {\overrightarrow{BC}}^{2}$$，即$$\left( \overrightarrow{AB} + \overrightarrow{AC} \right)^{2} > \left( \overrightarrow{AB} - \overrightarrow{AC} \right)^{2}$$，

所以$$\overrightarrow{AB} \cdot \overrightarrow{AC} > 0$$，又因为$$\angle A$$是$$\bigtriangleup ABC$$的内角，所以$$\angle A$$为锐角，故必要性也成立.

所以
"$$\angle A$$为锐角"是"$$\left| \overrightarrow{AB} + \overrightarrow{AC} \right| > \left| \overrightarrow{BC} \right|$$"的充分必要条件.

故选：A

7．已知$$\alpha + \beta = \frac{\text{π}}{4}$$，$$\text{tan}\alpha + \text{tan}\beta = 3$$，则$$\text{cos}(\alpha - \beta)$$的值为（    ）

> A．$$- \frac{\sqrt{2}}{6}$$ B．$$- \frac{\sqrt{2}}{2}$$
> C．$$\frac{\sqrt{2}}{2}$$ D．$$\frac{\sqrt{2}}{6}$$

【答案】A

【难度】0.65

【知识点】用和、差角的余弦公式化简、求值、用和、差角的正切公式化简、求值

【分析】由条件可得$$\tan(\alpha + \beta) = 1$$及$$\cos(\alpha + \beta) = \cos\frac{\text{π}}{4}$$，展开并联立可解得$$\cos\alpha\cos\beta,\sin\alpha\sin\beta$$，然后利用两角差的余弦公式求解.

【详解】∵$$\alpha + \beta = \frac{\text{π}}{4}$$，$$\text{tan}\alpha + \text{tan}\beta = 3$$，

∴$$\tan(\alpha + \beta) = \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta} = \frac{3}{1 - \tan\alpha\tan\beta} = 1$$，

∴$$\text{tan}\alpha\text{tan}\beta = - 2$$，即$$\frac{\sin\alpha\sin\beta}{\cos\alpha\cos\beta} = - 2$$，即$$\sin\alpha\sin\beta = - 2\cos\alpha\cos\beta$$①，

∵$$\cos(\alpha + \beta) = \cos\frac{\text{π}}{4}$$，∴$$\cos\alpha\cos\beta - \sin\alpha\sin\beta = \frac{\sqrt{2}}{2}$$②，

联立①②解得$$\cos\alpha\cos\beta = \frac{\sqrt{2}}{6},\sin\alpha\sin\beta = - \frac{\sqrt{2}}{3}$$，

∴$$\text{cos}(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta = \frac{\sqrt{2}}{6} - \frac{\sqrt{2}}{3} = - \frac{\sqrt{2}}{6}$$.

故选：A.

8．过点$$P\left( x_{0}，y_{0} \right)$$作曲线$$y = \text{ln}x$$的两条切线，记两切点分别为$$A\left( x_{1}，y_{1} \right)$$，$$B\left( x_{2}，y_{2} \right)$$，若两条切线斜率之积为$$1$$，则$$\frac{y_{0} + 1}{x_{0}}$$的取值范围是（    ）

> A．$$(0,1)$$ B．$$(0,1\rbrack$$ C．$$\lbrack 1, + \infty)$$
> D．$$(1, + \infty)$$

【答案】D

【难度】0.65

【知识点】求过一点的切线方程、基本不等式求和的最小值

【分析】利用导数的几何意义表示出切线方程，联立切线方程，求出$$x_{0}$$、$$y_{0}$$，再由两条切线的斜率之积为$$1$$得到$$x_{1}x_{2} = 1$$，即可用$$x_{1}$$的式子表示$$x_{0}$$、$$y_{0}$$，代入化简可得$$\frac{y_{0} + 1}{x_{0}} = \frac{1 + x_{1}^{2}}{2x_{1}}$$，利用基本不等式求解即可

【详解】因为$$y = \ln x$$
$$(x > 0)$$，所以$$y' = \frac{1}{x}$$，则$$y'|_{x = x_{1}} = \frac{1}{x_{1}}$$，$$y'|_{x = x_{2}} = \frac{1}{x_{2}}$$，

依题意可知两条切线的方程分别为$$y = \frac{1}{x_{1}}\left( x - x_{1} \right) + \ln x_{1}$$，$$y = \frac{1}{x_{2}}\left( x - x_{2} \right) + \ln x_{2}$$

联立两条切线的方程$$\left\{ \begin{array}{r}
y = \frac{1}{x_{1}}\left( x - x_{1} \right) + \ln x_{1} \\
y = \frac{1}{x_{2}}\left( x - x_{2} \right) + \ln x_{2}
\end{array} \right.\ $$

解得$$x_{0} = \frac{x_{1}x_{2}\left( \ln x_{2} - \ln x_{1} \right)}{x_{2} - x_{1}}$$，则$$y_{0} = \frac{x_{2}(\ln x_{2} - 1) - x_{1}(\ln x_{1} - 1)}{x_{2} - x_{1}}$$，

因为两条切线的斜率之积为$$1$$，所以$$\frac{1}{x_{1}x_{2}} = 1$$，所以$$x_{1}x_{2} = 1$$，则$$x_{2} = \frac{1}{x_{1}}$$

由$$x_{0} = \frac{x_{1}x_{2}\left( \ln x_{2} - \ln x_{1} \right)}{x_{2} - x_{1}}$$，$$y_{0} = \frac{x_{2}(\ln x_{2} - 1) - x_{1}(\ln x_{1} - 1)}{x_{2} - x_{1}}$$，

可得$$y_{0} + 1 = \frac{x_{2}(\ln x_{2} - 1) - x_{1}(\ln x_{1} - 1)}{x_{2} - x_{1}} + 1 = \frac{x_{2}\ln x_{2} - x_{1}\ln x_{1}}{x_{2} - x_{1}}$$

$$\frac{y_{0} + 1}{x_{0}} = \frac{\frac{x_{2}\ln x_{2} - x_{1}\ln x_{1}}{x_{2} - x_{1}}}{\frac{x_{1}x_{2}\left( \ln x_{2} - \ln x_{1} \right)}{x_{2} - x_{1}}} = \frac{x_{2}\ln x_{2} - x_{1}\ln x_{1}}{\ln x_{2} - \ln x_{1}} = \frac{\frac{1}{x_{1}}\ln\frac{1}{x_{1}} - x_{1}\ln x_{1}}{\ln\frac{1}{x_{1}} - \ln x_{1}} = \frac{1 + x_{1}^{2}}{2x_{1}}$$
$$\left( x_{1} > 0 \right)$$

所以$$\frac{y_{0} + 1}{x_{0}} = \frac{1 + x_{1}^{2}}{2x_{1}} = \frac{\frac{1}{x_{1}} + x_{1}}{2} \geq \frac{2\sqrt{\frac{1}{x_{1}} \cdot x_{1}}}{2} = 1$$，

当且仅当$$\frac{1}{x_{1}} = x_{1}$$，即$$x_{1} = 1$$时取得最小值，由因为$$x_{1} \neq x_{2} = \frac{1}{x_{1}}$$，所以$$x_{1} \neq 1$$，

则$$\frac{y_{0} + 1}{x_{0}} = \frac{1 + x_{1}^{2}}{2x_{1}} > 1$$，

故选：D

10．数学中有许多形状优美、寓意美好的曲线，如星形线、心形线、卵形线等、已知卵形线$$C：x^{2} + y - 2\sqrt{y} = 0$$，则（    ）
