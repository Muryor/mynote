1．已知集合$$A = \left\{ 1,2,3,4,5 \right\}$$，$$B = \left\{ x\left| 0 < \text{log}_{2}x < 2 \right.\  \right\}$$，则$$A \cap B =$$（    ）

> A．$$\left\{ 2,3 \right\}$$ B．$$\left\{ 2,3,4 \right\}$$
> C．$$\left\{ 1,2,3 \right\}$$ D．$$\left\{ 1,2,3,4 \right\}$$

【答案】A

【难度】0.85

【知识点】交集的概念及运算、由对数函数的单调性解不等式

【分析】先利用对数函数单调性解不等式，得到$$B = \left\{ x\left| 1 < x < 4 \right.\  \right\}$$，利用交集概念求出答案.

【详解】$$\text{log}_{2}x > 0 = \text{log}_{2}1$$，故$$x > 1$$，$$\text{log}_{2}x < 2 = \text{log}_{2}4$$，故$$x < 4$$，

所以$$B = \left\{ x\left| 1 < x < 4 \right.\  \right\}$$，

又$$A = \left\{ 1,2,3,4,5 \right\}$$，故$$A \cap B = \left\{ 2,3 \right\}$$.

故选：A.

【答案】D

【难度】0.65

【知识点】求函数值、函数奇偶性的应用

【分析】由奇函数的性质得出$$f(1) = 0$$，然后在等式$$f(x) - g(x - 2) = 2 - x$$中，分别令$$x = 1$$、$$x = 3$$，可得出$$g( - 1)$$的值，$$f(3)$$的值，由此可得$$f\left( g( - 1) \right)$$的值.

【详解】因为$$f(x + 1)$$是奇函数，则$$f( - x + 1) = - f(x + 1)$$，

18．已知函数$$f(x) = \text{sin}x$$，$$g(x) = \text{e}^{ax}$$，$$a \in R$$.

(1)若曲线$$y = f(x)$$在点$$O(0,0)$$的切线也是曲线$$y = g(x)$$的切线，求$$a$$的值；

(2)讨论函数$$h(x) = \frac{x - 1}{g(x)}$$在区间$$(0, + \infty)$$上的单调性；

(3)若$$f(x)g(x) < x$$对任意$$x \in (0, + \infty)$$恒成立，求$$a$$的取值范围.

【答案】(1)$$\frac{1}{\text{e}}$$

(2)答案见解析

(3)$$( - \infty,0\rbrack$$

【难度】0.4

【知识点】求在曲线上一点处的切线方程（斜率）、利用导数证明不等式、利用导数研究不等式恒成立问题、利用导数求函数（含参）的单调区间

【分析】（1）利用导数的几何意义来求曲线在某点处的切线方程即可求解；

（2）利用分类讨论，通过导数正负符号的判断可得单调区间；

（3）利用端点值刚好为$$0$$，要满足不等式恒成立，必要条件先行，再证明充分性即可求解.

【详解】（1）由已知得$$f'(x) = \text{cos}x$$，$$k = f'(0) = 1$$，$$\therefore f(x)$$在点$$O(0,0)$$处的切线方程为$$y = x$$.

设与$$y = g(x)$$切于$$P\left( x_{0},\text{e}^{ax_{0}} \right)$$，$$g'(x) = a\text{e}^{ax}$$，$$k = a\text{e}^{ax_{0}}$$，

则$$y = g(x)$$过该点的切线方程为：$$y - \text{e}^{ax_{0}} = a\text{e}^{ax_{0}}\left( x - x_{0} \right)$$，

整理得$$y = a\text{e}^{ax_{0}}x - a\text{e}^{ax_{0}}x_{0} + \text{e}^{ax_{0}}$$，由于该切线与$$y = x$$重合，

则$$\therefore\left\{ \begin{array}{r}
a\text{e}^{ax_{0}} = 1 \\
 - a\text{e}^{ax_{0}}x_{0} + \text{e}^{ax_{0}} = 0
\end{array} \Rightarrow \left\{ \begin{array}{r}
a = \frac{1}{\text{e}} \\
x_{0} = \text{e}
\end{array} \right.\  \right.\ $$.

（2）由$$h(x) = \frac{x - 1}{\text{e}^{ax}}$$，求导得$$h'(x) = \frac{\text{e}^{ax} - a\text{e}^{ax}(x - 1)}{\left( \text{e}^{ax} \right)^{2}} = \frac{a + 1 - ax}{\text{e}^{ax}}$$，

①当$$- 1 \leq a \leq 0$$时，$$\because x > 0$$，$$\therefore h'(x) > 0$$，$$h(x)$$在$$(0, + \infty)$$上单调递增；

②当$$a < - 1$$时，令$$h'(x) = 0 \Rightarrow x = \frac{a + 1}{a}$$

当$$0 < x < \frac{a + 1}{a}$$时，$$h'(x) < 0$$，$$h(x)$$在区间$$\left( 0,\frac{a + 1}{a} \right)$$上单调递减，

当$$x > \frac{a + 1}{a}$$时，$$h'(x) > 0$$，$$h(x)$$在区间$$\left( \frac{a + 1}{a}, + \infty \right)$$上单调递增

③当$$a > 0$$时，令$$h'(x) = 0 \Rightarrow x = \frac{a + 1}{a}$$

当$$0 < x < \frac{a + 1}{a}$$时，$$h'(x) > 0$$，$$h(x)$$在区间$$\left( 0,\frac{a + 1}{a} \right)$$上单调递增；

当$$x > \frac{a + 1}{a}$$时，$$h'(x) < 0$$，$$h(x)$$在区间$$\left( \frac{a + 1}{a}, + \infty \right)$$上单调递减

（3）由题意得$$\text{e}^{ax}\text{sin}x < x$$，即$$\text{e}^{ax}\text{sin}x - x < 0$$对$$\forall x \in (0, + \infty)$$恒成立.

令$$F(x) = \text{e}^{ax}\text{sin}x - x$$，$$F'(x) = a\text{e}^{ax}\text{sin}x + \text{e}^{ax}\text{cos}x - 1 = \text{e}^{ax}\left( a\text{sin}x + \text{cos}x - \text{e}^{- ax} \right)$$，

令$$\varphi(x) = a\text{sin}x + \text{cos}x - \text{e}^{- ax}$$，$$\varphi'(x) = a\text{cos}x - \text{sin}x + a\text{e}^{- ax}$$，

因为$$F(0) = \text{e}^{0}\text{sin}0 - 0 = 0$$，$$F'(0) = 0$$，

若$$\varphi'(0) = 2a > 0$$，则$$\varphi(x) = a\text{sin}x + \text{cos}x - \text{e}^{- ax}$$在$$x = 0$$处的切线必然是上升的，

又因为$$\varphi(0) = a\text{sin}0 + \text{cos}0 - \text{e}^{0} = 0$$，所以当$$x > 0$$且靠近$$0$$的函数值满足$$\varphi(x) > 0$$，

此时就有$$F'(x) = \text{e}^{ax}\left( a\text{sin}x + \text{cos}x - \text{e}^{- ax} \right) > 0$$，

从而可推导$$F(x) = \text{e}^{ax}\text{sin}x - x$$在$$x > 0$$且靠近$$0$$的附近是递增的，

又因为$$F(0) = \text{e}^{0}\text{sin}0 - 0 = 0$$，

所以在$$x > 0$$且靠近$$0$$的附近必有$$F(x) = \text{e}^{ax}\text{sin}x - x > 0$$

则必然不满足对$$\forall x \in (0, + \infty)$$恒有$$F(x) < 0$$，

所以要满足对$$\forall x \in (0, + \infty)$$恒有$$F(x) < 0$$，

首先必需满足在$$x > 0$$且靠近$$0$$的附近$$F'(x) \leq 0$$，

所以满足$$\varphi'(0) = 2a \leq 0 \Rightarrow a \leq 0$$，

从而可得参数$$a$$满足的必要条件是$$a \leq 0$$；

下面再证充分性，当$$a \leq 0$$，$$x > 0$$时，则$$\text{e}^{ax} < \text{e}^{0} = 1$$，即有$$\frac{x}{\text{e}^{ax}} > x$$，

又构造$$\varphi(x) = x - \sin x$$，$$x > 0$$，可得$$\varphi'(x) = 1 - \cos x \geq 0$$，

所以$$\varphi(x) = x - \sin x$$在区间$$(0, + \infty)$$上单调递增，即$$\varphi(x) = x - \sin x > \varphi(0) = 0$$，

则可知$$\text{sin}x - \frac{x}{\text{e}^{ax}} \leq \text{sin}x - x < 0$$，则$$F(x) = \text{e}^{ax}\left( \text{sin}x - \frac{x}{\text{e}^{ax}} \right) < 0$$，

$$\therefore F(x) < 0$$恒成立，符合题意，

综上：$$a$$的取值范围为$$( - \infty,0\rbrack$$.

【点睛】方法点睛：针对端点值刚好是不等式的临界值时，利用必要条件先确定参数的范围，再进行充分性证明，如若成立，则这个必要条件就是充要条件.
