1．已知集合$$A = \left\{ x \mid x^{2} - 5x + 4 < 0 \right\}$$
$$B = \left\{ n\left| \frac{2n}{n - 1} \right.\  \right.\ $$是质数，$$\left. \ n \in \text{N} \right\}$$，则$$A \cap B =$$（    ）

> A．$$\varnothing$$ B．{2} C．{3} D．$$\left\{ 2，3 \right\}$$

【答案】C

【难度】0.65

【知识点】列举法表示集合、交集的概念及运算、解不含参数的一元二次不等式

【分析】通过解不等式先求出集合$$A$$，变形$$\frac{2n}{n - 1} = 2 + \frac{2}{n - 1}$$，分析出要使$$\frac{2n}{n - 1}\left( n \in \text{N},n \neq 1 \right)$$是质数，而$$n - 1$$必须是2的正因数，将$$n - 1 = 1$$和$$n - 1 = 2$$分别代入验证，即可求出集合$$B$$，再求$$A \cap B$$即可得解.

【详解】由$$x^{2} - 5x + 4 = (x - 1)(x - 4) < 0$$，解得$$1 < x < 4$$，故$$A = \left\{ x\left| 1 < x < 4 \right.\  \right\}$$.

因为$$\frac{2n}{n - 1} = \frac{2(n - 1) + 2}{n - 1} = 2 + \frac{2}{n - 1}\left( n \in \text{N},n \neq 1 \right)$$，

要使$$\frac{2n}{n - 1}$$是质数，$$\frac{2}{n - 1}$$必须是整数，而$$n - 1$$必须是2的正因数.

因为2的正因数有1和2，

所以当$$n - 1 = 1$$时，$$n = 2$$，此时$$\frac{2n}{n - 1} = \frac{2 \times 2}{2 - 1} = 4$$，4不是质数，不符合要求，舍去；

所以当$$n - 1 = 2$$时，$$n = 3$$，此时$$\frac{2n}{n - 1} = \frac{2 \times 3}{3 - 1} = 3$$，3是质数，符合要求，故$$B = \left\{ 3 \right\}$$.

所以$$A \cap B = \left\{ 3 \right\}$$.

故选：C

7．若实数$$x = 10$$，$$y = 5\text{eln}2$$，$$z = 2\text{eln}5$$则$$x,y,z$$的大小关系是（    ）

> A．$$x > y > z$$ B．$$x > z > y$$ C．$$y > x > z$$ D．$$y > z > x$$

【答案】A

【难度】0.65

【知识点】函数单调性、极值与最值的综合应用、比较对数式的大小

【分析】先利用对数函数的单调性，比较$$y$$与$$z$$的大小，再构造函数$$f(x) = \frac{\ln x}{x}$$，$$x > 0$$，分析其单调性和最值，比较$$x$$与$$y$$的大小.

【详解】因为$$y = 5\text{eln}2 = \text{eln}2^{5} = \text{eln32}$$，$$z = 2\text{eln}5 = \text{eln}5^{2} = \text{eln}25$$，

由$$\ln 32 > \ln 25 > 0$$，所以$$\text{e}\ln 32 > \text{e}\ln 25$$，即$$y > z$$.

设函数$$f(x) = \frac{\ln x}{x}$$，$$x > 0$$，则$$f'(x) = \frac{1 - \ln x}{x^{2}}$$，$$x > 0$$.

由$$f'(x) > 0$$ $$\Rightarrow$$ $$0 < x < \text{e}$$；由$$f'(x) < 0$$
$$\Rightarrow$$ $$x > \text{e}$$.

即$$f(x)$$在$$\left( 0,\text{e} \right)$$上单调递增，在$$\left( \text{e}, + \infty \right)$$上单调递减.

所以$$f(x) \leq f\left( \text{e} \right) = \frac{1}{\text{e}}$$，所以$$\text{e}f(x) \leq 1$$.

所以$$\text{e}f(2) < 1$$ $$\Rightarrow$$ $$\frac{\text{e}\ln 2}{2} < 1$$
$$\Rightarrow$$ $$\text{e}\ln 2 < 2$$ $$\Rightarrow$$
$$5\text{e}\ln 2 < 10$$，即$$y < x$$.

综上，$$x > y > z$$.

故选：A

8．已知函数$$f(x) = x^{3} + 3ax^{2} + x\left( a \in \text{R} \right),P$$是函数$$f(x)$$的图象上的定点，过$$P$$的动直线与函数$$f(x)$$的图象有异于$$P$$的两个公共点$$M,N$$，且它们的纵坐标之和恒为2，则$$P$$的横坐标为（    ）

> A．1 B．$$- 1$$ C．2 D．$$- 2$$

【答案】B

【难度】0.4

【知识点】二次函数的图象分析与判断

【分析】设$$P\left( x_{0},x_{0}^{3} + 3ax_{0}^{2} + x_{0} \right)$$，$$M\left( x_{1},y_{1} \right)$$，$$N\left( x_{2},y_{2} \right)$$，直线$$MN$$的方程为$$y = k\left( x - x_{0} \right) + \left( x_{0}^{3} + 3ax_{0}^{2} + x_{0} \right)$$，与函数解析式联立，利用$$y_{1} + y_{2} = 2$$恒成立求$$x_{0}$$.

【详解】因为$$P$$在函数$$f(x)$$的图象上，所以可设$$P\left( x_{0},x_{0}^{3} + 3ax_{0}^{2} + x_{0} \right)$$，

设直线$$MN$$方程为：$$y - \left( x_{0}^{3} + 3ax_{0}^{2} + x_{0} \right) = k\left( x - x_{0} \right)$$
$$\Rightarrow$$
$$y = k\left( x - x_{0} \right) + \left( x_{0}^{3} + 3ax_{0}^{2} + x_{0} \right)$$，

代入$$y = x^{3} + 3ax^{2} + x$$得：$$x^{3} + 3ax^{2} + x = k\left( x - x_{0} \right) + \left( x_{0}^{3} + 3ax_{0}^{2} + x_{0} \right)$$，

化简得：$$x^{3} + 3ax^{2} + (1 - k)x - x_{0}^{3} - 3ax_{0}^{2} - (1 - k)x_{0} = 0$$.

因为$$x = x_{0}$$为该方程的1个根，所以方程可化成$$\left( x - x_{0} \right)\left( x^{2} + bx + c \right) = 0$$，

即$$x^{3} + \left( b - x_{0} \right)x^{2} + \left( c - bx_{0} \right)x - cx_{0} = 0$$.

所以$$b - x_{0} = 3a$$ $$\Rightarrow$$ $$b = x_{0} + 3a$$.

设$$M\left( x_{1},y_{1} \right)$$，$$N\left( x_{2},y_{2} \right)$$，

则$$x_{1}$$，$$x_{2}$$为方程$$x^{2} + bx + c = 0$$的两根，所以$$x_{1} + x_{2} = - b = - x_{0} - 3a$$，

由$$y_{1} + y_{2} = 2$$ $$\Rightarrow$$
$$k\left( x_{1} - x_{0} \right) + \left( x_{0}^{3} + 3ax_{0}^{2} + x_{0} \right) + k\left( x_{2} - x_{0} \right) + \left( x_{0}^{3} + 3ax_{0}^{2} + x_{0} \right) = 2$$，

即$$k\left( x_{1} + x_{2} - 2x_{0} \right) + 2\left( x_{0}^{3} + 3ax_{0}^{2} + x_{0} \right) = 2$$恒成立.

所以$$\{\begin{array}{r}
x_{1} + x_{2} - 2x_{0} = 0 \\
2(x_{0}^{3} + 3ax_{0}^{2} + x_{0}) = 2
\end{array}$$，

由$$x_{1} + x_{2} - 2x_{0} = 0$$及$$x_{1} + x_{2} = - x_{0} - 3a$$可得$$( - x_{0} - 3a) - 2x_{0} = 0$$，解得$$x_{0} = - a$$，

将$$x_{0} = - a$$代入该式得$$( - a)^{3} + 3a( - a)^{2} + ( - a) = 1$$，即$$2a^{3} - a - 1 = 0$$，

$$(a - 1)\left( 2a^{2} + 2a + 1 \right) = 0$$，

所以$$a = 1$$，所以$$x_{0} = - 1$$，即$$P$$点的横坐标为：$$- 1$$.

故选：B

所以$$f(x)$$在区间$$\left\lbrack \frac{\pi}{2},\frac{3\pi}{4} \right\rbrack$$上单调递减．

$$f{(x)}_{\min} = f\left( \frac{3\pi}{4} \right) = \ln\left( \frac{3\pi}{4} + 1 \right) - \frac{\sqrt{2}}{2}$$.

因为$$\frac{3\pi + 4}{4} > 3 > e,\ln\left( \frac{3\pi + 4}{4} \right) > \ln e = 1 > \frac{\sqrt{2}}{2}$$，所以$$f\left( \frac{3\pi}{4} \right) > 0$$，在$$\left\lbrack \frac{\pi}{2},\frac{3\pi}{4} \right\rbrack$$上$$f(x) > 0$$，故该区间内无零点．

④当$$x \in \left\lbrack \frac{3\pi}{4}, + \infty \right)$$时，$$f(x) = \ln(x + 1) + \cos x \geq \ln(x + 1) - 1 \geq \ln\left( \frac{3\pi}{4} + 1 \right) - 1 > 0$$，

因此该区间内也无零点．

综上，函数$$f(x)$$在$$( - 1, + \infty)$$上仅有一个零点．

（3）函数$$f(x) = \ln(x + 1) + a\cos x(a \in Z)$$，由题意$$f(0) = a \geq 0$$．

当$$a = 0$$时，$$f(x) = \ln(x + 1) \geq 0$$对于任意$$x \geq 0$$恒成立，

当$$a = 1$$时，由（2）知$$f(x) \geq 0$$在$$\lbrack 0, + \infty)$$上恒成立．

当$$a \geq 2$$且$$a \in Z$$时，取$$x = \pi$$，有$$f(\pi) = \ln(\pi + 1) + a\cos\pi = \ln(\pi + 1) - a$$，

由于$$\ln(\pi + 1) < 2$$且$$a \geq 2$$，故$$f(\pi) < 0$$，与$$f(x) \geq 0$$矛盾．

综上所述，$$a$$的值为0或1．
