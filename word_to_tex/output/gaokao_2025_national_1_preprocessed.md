【难度】0.15

【知识点】由导数求函数的最值（不含参）、解余弦不等式

【分析】（1）利用导数结合三角变换得导数零点，讨论导数的符号后得单调性，从而可求最大值；或者利用均值不等式可求最大值.

（2）利用反证法可证三角不等式有解；

（3）先考虑$$\varphi = 0,\text{π}$$时$$b$$的范围，对于$$\varphi \in \left( 0,\text{π} \right)$$时，可利用（2）中的结论结合特值法求得$$b \geq 3\sqrt{3}$$，从而可得$$b$$的最小值；或者先根据函数解析特征得$$b \geq 0$$，再结合特值法可得$$b \geq 3\sqrt{3}$$，结合（1）的结果可得$$b$$的最小值.

【详解】（1）法1：$$f'(x) = - 5\sin x + 5\sin 5x = 10\cos 3x\sin 2x$$，

因为$$x \in \left\lbrack 0,\frac{\text{π}}{4} \right\rbrack$$，故$$2x \in \left\lbrack 0,\frac{\text{π}}{2} \right\rbrack$$，故$$\sin 2x \geq 0$$，

当$$0 < x < \frac{\text{π}}{6}$$时，$$\cos 3x > 0$$即$$f'(x) > 0$$，

当$$\frac{\text{π}}{6} < x < \frac{\text{π}}{4}$$时，$$\cos 3x < 0$$即$$f'(x) < 0$$，

故$$f(x)$$在$$\left( 0,\frac{\text{π}}{6} \right)$$上为增函数，在$$\left( \frac{\text{π}}{6},\frac{\text{π}}{4} \right)$$为减函数，

故$$f(x)$$在$$\left\lbrack 0,\frac{\text{π}}{4} \right\rbrack$$上的最大值为$$f\left( \frac{\text{π}}{6} \right) = 5\cos\frac{\text{π}}{6} - \cos\frac{\text{5π}}{6} = 3\sqrt{3}$$.

法2：我们有$$\cos 5x = \cos(x + 4x) = \cos x\cos 4x - \sin x\sin 4x$$

$$= \cos x\left( 2\cos^{2}2x - 1 \right) - \sin x \cdot 2\sin 2x\cos 2x$$

$$= \cos x\left( 2\left( 2\cos^{2}x - 1 \right)^{2} - 1 \right) - \sin x \cdot 2 \cdot 2\sin x\cos x\cos 2x$$

$$= \cos x\left( 8\cos^{4}x - 8\cos^{2}x + 1 \right) - 4\cos x\cos 2x\sin^{2}x$$

$$= 8\cos^{5}x - 8\cos^{3}x + \cos x - 4\cos x\left( 2\cos^{2}x - 1 \right)\left( 1 - \cos^{2}x \right)$$

$$= 16\cos^{5}x - 20\cos^{3}x + 5\cos x$$.

所以：

$$f(x) = 5\cos x - \cos 5x = 5\cos x - \left( 16\cos^{5}x - 20\cos^{3}x + 5\cos x \right) = 20\cos^{3}x - 16\cos^{5}x$$

$$= 4\cos^{3}x\left( 5 - 4\cos^{2}x \right) \leq 4\left| \cos x \right|^{3}\left( 5 - 4\left| \cos x \right|^{2} \right) = 4\left| \cos x \right|^{3}\left( \sqrt{5} - 2\left| \cos x \right| \right)\left( \sqrt{5} + 2\left| \cos x \right| \right)$$

$$= \frac{32}{3}\left( \left| \cos x \right| \cdot \left| \cos x \right| \cdot \left| \cos x \right| \cdot \frac{\sqrt{15} + 3}{4}\left( \sqrt{5} - 2\left| \cos x \right| \right) \cdot \frac{\sqrt{15} - 3}{4}\left( \sqrt{5} + 2\left| \cos x \right| \right) \right)$$

$$\leq \frac{32}{3}\left( \frac{1}{5}\left( \left| \cos x \right| + \left| \cos x \right| + \left| \cos x \right| + \frac{\sqrt{15} + 3}{4}\left( \sqrt{5} - 2\left| \cos x \right| \right) + \frac{\sqrt{15} - 3}{4}\left( \sqrt{5} + 2\left| \cos x \right| \right) \right) \right)^{5}$$

$$= \frac{32}{3}\left( \frac{\sqrt{3}}{2} \right)^{5} = 3\sqrt{3}$$.

这得到$$f(x) \leq 3\sqrt{3}$$，同时又有$$f\left( \frac{\text{π}}{6} \right) = 5\cos\frac{\text{π}}{6} - \cos\frac{\text{5π}}{6} = 3\sqrt{3}$$，

故$$f(x)$$在$$\left\lbrack 0,\frac{\text{π}}{4} \right\rbrack$$上的最大值为$$3\sqrt{3}$$，在$$R$$上的最大值也是$$3\sqrt{3}$$.

（2）法1：由余弦函数的性质得$$\cos x \leq \cos\theta$$的解为$$\left\lbrack 2k\text{π} + \theta,2k\text{π} + \text{2π} - \theta \right\rbrack$$，$$k \in \text{Z}$$，

若任意$$\left\lbrack 2k\text{π} + \theta,2k\text{π+2π} - \theta \right\rbrack,k \in \text{Z}$$与$$\lbrack a - \theta,a + \theta\rbrack$$交集为空，

则$$a - \theta > 2k\text{π} + 2\text{π} - \theta$$且$$a + \theta < 2k\text{π} + 2\text{π} + \theta$$，此时$$a$$无解，

矛盾，故无解；故存在$$k \in \text{Z}$$，使得$$\left\lbrack 2k\text{π} - \theta\text{,}2k\text{π} + \theta \right\rbrack \cap (a - \theta,a + \theta) \neq \varnothing$$，

法2：由余弦函数的性质知$$\cos y \leq \cos\theta$$的解为$$\left\lbrack 2k\text{π} + \theta\text{,}2(k + 1)\text{π} - \theta \right\rbrack(k \in Z)$$，

若每个$$\left\lbrack 2k\text{π} + \theta\text{,}2(k + 1)\text{π} - \theta \right\rbrack$$与$$\lbrack a - \theta,a + \theta\rbrack$$交集都为空，

则对每个$$k \in Z$$，必有$$2(k + 1)\text{π} - \theta < a - \theta$$或$$2k\text{π} + \theta > a + \theta$$之一成立.

此即$$k < \frac{a}{2\text{π}} - 1$$或$$k > \frac{a}{2\text{π}}$$，但长度为$$1$$的闭区间$$\left\lbrack \frac{a}{2\text{π}} - 1,\frac{a}{2\text{π}} \right\rbrack$$上必有一整数$$k$$，

该整数$$k$$不满足条件，矛盾.

故存在$$y \in \lbrack a - \theta,a + \theta\rbrack$$，使得$$\cos y \leq \cos\theta$$成立.

（3）法1：记$$h(x) = 5\cos x - \cos(5x + \varphi)$$，

因为$$h(x + 2\text{π}) = 5\cos\left( x + 2\text{π} \right) - \cos\left( 5x + 10\text{π+}\varphi \right) = h(x)$$，

故$$h(x)$$为周期函数且周期为$$2\text{π}$$,故只需讨论$$x \in \left\lbrack 0,2\text{π} \right\rbrack,\varphi \in \left\lbrack 0,\text{π} \right\rbrack$$的情况.

当$$\varphi = \text{π}$$时，$$h(x) = 5\cos x - \cos\left( 5x + \text{π} \right) = 5\cos x + \cos 5x \leq 6$$，

当$$\varphi = \text{0}$$时，$$h(x) = 5\cos x - \cos 5x$$，

此时$$h'(x) = - 5\sin x + 5\sin 5x = 10\cos 3x\sin 2x,x \in \left( 0,2\text{π} \right)$$，

令$$h'(x) = 0$$，则$$x = \frac{\text{π}}{6},\frac{\text{π}}{2},\frac{\text{5π}}{6},\text{π},\frac{\text{7π}}{6},\frac{\text{3π}}{2},\frac{\text{11π}}{6}$$，

而$$h(\frac{\text{π}}{6}) = h(\frac{\text{11π}}{6}) = 3\sqrt{3},h(\frac{\text{π}}{2}) = h(\frac{\text{3π}}{2}) = 0,h(\frac{\text{5π}}{6}) = h(\frac{\text{7π}}{6}) = - 3\sqrt{3},h(\text{π}) = - 4$$，

$$h(0) = h(2\text{π}) = 4$$，故$$h{(x)}_{\max} = h(\frac{\text{π}}{6}) = h(\frac{\text{11π}}{6}) = 3\sqrt{3}$$，

当$$\varphi \in \left( 0,\text{π} \right)$$，在（2）中取$$\varphi = a$$，则存在$$y \in \lbrack\varphi - \theta,\varphi + \theta\rbrack$$，使得$$\cos y \leq \cos\theta$$，

取$$\theta = \frac{5\text{π}}{6}$$，则$$\cos y \leq - \frac{\sqrt{3}}{2}$$，取$$x = \frac{y - \varphi}{5} \in \left( - \frac{\theta}{5},\frac{\theta}{5} \right)$$即$$x = \frac{y - \varphi}{5} \in \left( - \frac{\pi}{6},\frac{\pi}{6} \right)$$，

故$$5\cos x \geq \frac{5\sqrt{3}}{2}$$，故$$5\cos x - \cos(5x + \varphi) \geq 3\sqrt{3}$$，

综上$$b \geq 3\sqrt{3}$$，可取$$x = \frac{\text{π}}{6}$$，$$\varphi = 0$$使得等号成立.

综上，$$b_{\min} = 3\sqrt{3}$$.

法2：设$$g_{\varphi}(x) = 5\cos x - \cos(5x + \varphi)$$.

①一方面，若存在$$\varphi$$，使得$$g_{\varphi}(x) = 5\cos x - \cos(5x + \varphi) \leq b$$对任意$$x$$恒成立，

则对这样的$$\varphi$$，同样有$$g_{\varphi}(x) = - g_{\varphi}\left( x + \text{π} \right) \geq - b$$.

所以$$\left| g_{\varphi}(x) \right| \leq b$$对任意$$x$$恒成立，这直接得到$$b \geq 0$$.

设$$\frac{\varphi}{6} - \frac{\text{π}}{6} = m$$，则根据$$\left| g_{\varphi}(x) \right| \leq b$$恒成立，有

$$b \geq \left| g_{\varphi}\left( - \frac{\varphi}{6} + \frac{\text{π}}{6} \right) \right| = \left| 5\cos\left( - \frac{\varphi}{6} + \frac{\text{π}}{6} \right) - \cos\left( \frac{\varphi}{6} + \frac{\text{5π}}{6} \right) \right| = \left| 5\cos\left( \frac{\varphi}{6} - \frac{\text{π}}{6} \right) + \cos\left( \frac{\varphi}{6} - \frac{\text{π}}{6} \right) \right| = \left| 6\cos\left( \frac{\varphi}{6} - \frac{\text{π}}{6} \right) \right| = 6\left| \cos m \right|$$

$$b \geq \left| g_{\varphi}\left( - \frac{\varphi}{6} - \frac{\text{π}}{6} \right) \right| = \left| 5\cos\left( - \frac{\varphi}{6} - \frac{\text{π}}{6} \right) - \cos\left( \frac{\varphi}{6} - \frac{\text{5π}}{6} \right) \right| = \left| 5\cos\left( \frac{\varphi}{6} + \frac{\text{π}}{6} \right) + \cos\left( \frac{\varphi}{6} + \frac{\text{π}}{6} \right) \right| = \left| 6\cos\left( \frac{\varphi}{6} + \frac{\text{π}}{6} \right) \right| = 6\left| \cos\left( m + \frac{\text{π}}{3} \right) \right|$$

$$b \geq \left| g_{\varphi}\left( - \frac{\varphi}{6} + \frac{\text{π}}{2} \right) \right| = \left| 5\cos\left( - \frac{\varphi}{6} + \frac{\text{π}}{2} \right) - \cos\left( \frac{\varphi}{6} + \frac{\text{5π}}{2} \right) \right| = \left| 5\cos\left( \frac{\varphi}{6} - \frac{\text{π}}{2} \right) + \cos\left( \frac{\varphi}{6} - \frac{\text{π}}{2} \right) \right| = \left| 6\cos\left( \frac{\varphi}{6} - \frac{\text{π}}{2} \right) \right| = 6\left| \cos\left( m - \frac{\text{π}}{3} \right) \right|$$

所以$$\left| \cos m \right|,\left| \cos\left( m + \frac{\text{π}}{3} \right) \right|,\left| \cos\left( m - \frac{\text{π}}{3} \right) \right|$$均不超过$$\frac{b}{6}$$，

再结合$$\cos 2x = 2\left| \cos x \right|^{2} - 1$$，

就得到$$\cos 2m,\cos\left( 2m + \frac{\text{2π}}{3} \right),\cos\left( 2m - \frac{\text{2π}}{3} \right)$$均不超过$$2\left( \frac{b}{6} \right)^{2} - 1 = \frac{b^{2}}{18} - 1$$.

假设$$b < 3\sqrt{3}$$，则$$\frac{b^{2}}{18} - 1 < \frac{\left( 3\sqrt{3} \right)^{2}}{18} - 1 = \frac{1}{2}$$，

故$$\cos 2m,\cos\left( 2m + \frac{\text{2π}}{3} \right),\cos\left( 2m - \frac{\text{2π}}{3} \right) \in \left\lbrack - 1,\frac{1}{2} \right)$$.

但这是不可能的，因为三个角$$2m,2m + \frac{\text{2π}}{3},2m - \frac{\text{2π}}{3}$$和单位圆的交点将单位圆三等分，

这三个点不可能都在直线$$x = \frac{1}{2}$$左侧.

所以假设不成立，这意味着$$b \geq 3\sqrt{3}$$.

②另一方面，若$$b = 3\sqrt{3}$$，则由（1）中已经证明$$f(x) \leq 3\sqrt{3}$$，

知存在$$\varphi = 0$$，使得$$5\cos x - \cos(5x + \varphi) = 5\cos x - \cos 5x = f(x) \leq 3\sqrt{3} = b$$.

从而$$b = 3\sqrt{3}$$满足题目要求.

综合上述两个方面，可知$$b$$的最小值是$$3\sqrt{3}$$.
