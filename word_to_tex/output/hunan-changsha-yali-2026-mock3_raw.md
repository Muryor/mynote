1．若复数$$z$$满足$$\left( 1 + 2\text{i} \right)z = 4 + 3\text{i}$$，则$$\overline{z}$$的实部为（　　）

> A．1 B．－1 C．2 D．－2

【答案】C

【难度】0.65

【知识点】求复数的实部与虚部、复数的除法运算

【分析】先用除法运算得出复数$$z$$，写出它的共轭复数$$\overline{z}$$即可找出$$\overline{z}$$的实部.

【详解】因为$$\left( 1 + 2\text{i} \right)z = 4 + 3\text{i}$$，

所以$$z = \frac{4 + 3\text{i}}{1 + 2\text{i}} = \frac{\left( 4 + 3\text{i} \right)\left( 1 - 2\text{i} \right)}{\left( 1 + 2\text{i} \right)\left( 1 - 2\text{i} \right)} = \frac{10 - 5\text{i}}{5} = 2 - \text{i}$$，

所以$$\overline{z} = 2 + \text{i}$$，

所以$$\overline{z}$$的实部为2，

故选：C.

6．已知奇函数$$f(x)$$的定义域为$$R$$，且函数$$y = f(x)$$图象关于$$x = 2$$对称．当$$x \in \lbrack 0，2\rbrack$$时，$$f(x) = x$$，则$$f(13) =$$（　　）

> A．1 B．-1 C．2 D．-2

【答案】B

【难度】0.65

【知识点】函数奇偶性的应用、函数对称性的应用、由函数的周期性求函数值

【分析】利用奇函数以及对称性求出函数周期为$$8$$，所以$$f(13) = f(5) = f( - 3) = - f(3) = - f(1)$$，即可求解.

【详解】因为函数$$y = f(x)$$图象关于$$x = 2$$对称，所以$$f(2 + x) = f(2 - x)$$，

因为$$f(x)$$为奇函数，所以$$f( - x) = - f(x)$$，

所以$$f(2 - x) = - f(x - 2)$$，即$$f(x + 2) = - f(x - 2)$$，

所以$$f(x + 4) = - f(x)$$，所以$$f(x + 8) = - f(x + 4) = f(x)$$，所以$$f(x)$$的周期为8，

所以$$f(13) = f(5 + 8) = f(5) = f( - 3 + 8) = f( - 3) = - f(3)$$，

而$$f(3) = f(2 + 1) = f(2 - 1) = f(1)$$，

又因为当$$x \in \lbrack 0，2\rbrack$$时，$$f(x) = x$$，所以$$f(1) = 1$$，即$$f(3) = f(1) = 1$$，

所以$$f(13) = - f(3) = - f(1) = - 1$$.

故选：B.

7．志愿者甲参加第$$21$$届文博会的服务工作，甲从住所到文博会选择乘地铁、乘公交车、骑共享单车的概率分别为$$\frac{1}{4}$$，$$\frac{1}{4}$$，$$\frac{1}{2}$$，且乘地铁、乘公交车、骑共享单车按时到达文博会的概率分别为$$\frac{4}{5}$$，$$\frac{3}{4}$$，$$\frac{2}{3}$$．若某一天甲按时到达文博会，则他骑共享单车的概率为（    ）

> A．$$\frac{173}{240}$$ B．$$\frac{93}{173}$$ C．$$\frac{80}{173}$$
> D．$$\frac{1}{3}$$

【答案】C

【难度】0.85

【知识点】计算条件概率、利用全概率公式求概率

【分析】根据全概率公式及条件概率公式直接求解.

【详解】设事件$$A$$表示"甲乘地铁"，事件$$B$$表示"甲乘公交车"，事件$$C$$表示"甲骑共享单车"，事件$$D$$表示"甲按时到达文博会"，

则$$P(A) = \frac{1}{4}$$，$$P(B) = \frac{1}{4}$$，$$P(C) = \frac{1}{2}$$，$$P\left( \left. \ D \right|A \right) = \frac{4}{5}$$，$$P\left( \left. \ D \right|B \right) = \frac{3}{4}$$，$$P\left( \left. \ D \right|C \right) = \frac{2}{3}$$，

则$$P(D) = P(A)P\left( \left. \ D \right|A \right) + P(B)P\left( \left. \ D \right|B \right) + P(C) \cdot P\left( \left. \ D \right|C \right)$$

$$= \frac{1}{4} \times \frac{4}{5} + \frac{1}{4} \times \frac{3}{4} + \frac{1}{2} \times \frac{2}{3} = \frac{173}{240}$$，

$$P(CD) = P(C)P\left( D\left| C \right.\  \right) = \frac{1}{3}$$，

所以若某一天甲按时到达文博会，

则他骑共享单车的概率为$$P\left( \left. \ C \right|D \right) = \frac{P(CD)}{P(D)} = \frac{80}{173}$$．

8．已知直线$$l:2x\text{cos}\alpha + 2y\text{sin}\alpha - 1 = 0$$与圆$$C:(x - 1)^{2} + \left( y - \sqrt{3} \right)^{2} = \frac{9}{4}$$相切，则满足条件的直线有（    ）

> A．1条 B．2条 C．3条 D．4条

【答案】C

【难度】0.65

【知识点】由直线与圆的位置关系求参数、圆的公切线条数

【分析】法一：借助点到直线距离公式可得原点$$O$$到直线$$l$$的距离为定值，则可得直线$$l$$与圆$$O:x^{2} + y^{2} = \frac{1}{4}$$相切，再结合公切线性质计算即可得解；法二：由公切线性质可得圆心$$C$$到直线$$l$$的距离为半径，则可得$$\alpha$$取值可能，即可得解.

【详解】法一：原点$$O$$到直线$$l$$的距离为$$d = \frac{1}{\sqrt{4\text{cos}^{2}\alpha + 4\text{sin}^{2}\alpha}} = \frac{1}{2}$$，

从而满足条件的直线$$l$$为圆$$O:x^{2} + y^{2} = \frac{1}{4}$$和圆$$C:(x - 1)^{2} + \left( y - \sqrt{3} \right)^{2} = \frac{9}{4}$$的公切线，

因为$$|OC| = 2 = \frac{1}{2} + \frac{3}{2}$$，故两圆外切，共有三条公切线．

法二：由题设知，圆心$$C$$到直线$$l$$的距离为：

$$d = \frac{\left| 2\text{cos}\alpha + 2\sqrt{3}\text{sin}\alpha - 1 \right|}{\sqrt{4\text{cos}^{2}\alpha + 4\text{sin}^{2}\alpha}} = \frac{\left| 4\text{sin}\left( \alpha + \frac{\text{π}}{6} \right) - 1 \right|}{2} = \frac{3}{2}$$，

化简得$$4\text{sin}\left( \alpha + \frac{\text{π}}{6} \right) - 1 = \pm 3$$，即$$\text{sin}\left( \alpha + \frac{\text{π}}{6} \right) = 1$$或$$- \frac{1}{2}$$，

当$$\text{sin}\left( \alpha + \frac{\text{π}}{6} \right) = 1$$时，$$\alpha = \frac{\text{π}}{3} + 2k\text{π},k \in Z$$；

当$$\text{sin}\left( \alpha + \frac{\text{π}}{6} \right) = - \frac{1}{2}$$时，$$\alpha = \frac{5\text{π}}{3} + 2k\text{π}$$或$$\text{π} + 2k\text{π},k \in Z$$．

故满足条件的直线有三条．

> A．$$|\overrightarrow{a}| = |\overrightarrow{b}|$$
> B．$$\overrightarrow{a}$$与$$\overrightarrow{b}$$的夹角为$$\frac{3}{4}\text{π}$$
>

【难度】0.85

【知识点】向量垂直的坐标表示、向量夹角的坐标表示、利用坐标求向量的模、求投影向量

【分析】根据平面向量数量积的坐标运算逐项判断.

【详解】对于A：$$\left| \overrightarrow{a} \right| = \sqrt{1^{2} + 1^{2}} = \sqrt{2},\left| \overrightarrow{b} \right| = \sqrt{( - 2)^{2} + 0^{2}} = 2$$，故A错误.

对于B：$$\cos\left\langle \overrightarrow{a},\overrightarrow{b} \right\rangle = \frac{\overrightarrow{a} \cdot \overrightarrow{b}}{\left| \overrightarrow{a} \right|\left| \overrightarrow{b} \right|} = \frac{- 2}{2\sqrt{2}} = - \frac{\sqrt{2}}{2}$$，因为$$\left\langle \overrightarrow{a},\overrightarrow{b} \right\rangle \in \left\lbrack 0,\text{π} \right\rbrack$$，所以$$\left\langle \overrightarrow{a},\overrightarrow{b} \right\rangle = \frac{3\text{π}}{4}$$，故B正确；

故选：BCD.
