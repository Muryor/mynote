\begin{question}
已知\(\triangle ABC\)的内角\(A\)，\(B\)，
\(C\)所对的边分别为\(a\)，\(b\)，\(c\)，

  \item 当\(B\)最大时，\(\sin C - \sin B = \sin B\cos A\)成立
  \item 若\(c = 2a\)，则\(C > \frac{2\pi}{3}\)

\topics{解余弦不等式；余弦定理解三角形；正弦定理解三角形}
\difficulty{0.65}
\answer{AD}
\explain{A选项，
\(\cos B = \frac{a^{2} + c^{2} - b^{2}}{2ac} = \frac{a^{2} + c^{2} - ac}{2ac} \geq \frac{2ac - ac}{2ac} = \frac{1}{2}\)，\par

所以\(B \in \left( 0,\frac{\pi}{3} \right\rbrack\)，
A正确；\par
B选项，当\(B = \frac{\pi}{3}\)时，\(a = c\)，

(ii)记\(C\)关于原点对称的点为\(D\)，证明：\(A\)，\(B\)，\(C\)，\(D\)四点共圆.
\end{enumerate}
\topics{椭圆中的定值问题；根据a；b；c求椭圆标准方程；椭圆中三角形(四边形)的面积}
\difficulty{0.4}
\answer{(1)\(\frac{x^{2}}{2} + y^{2} = 1\)
(2)(i)\(\frac{\sqrt{6}}{2}\)；(ii)证明见解析}
\explain{(1)由题意，
当\(P\)在短轴顶点时\(S_{\triangle OFP}\)最大，
此时\(S_{\triangle OFP} = \frac{1}{2}\sqrt{a^{2} - 1} = \frac{1}{2}\)，\par
解得\(a^{2} = 2\)，
所以椭圆方程为：\(\frac{x^{2}}{2} + y^{2} = 1\)；\par
(2)(i)由题意可得，直线\(l\)的斜率存在，
设\(A(x_{1},y_{1})\)，\(B(x_{2},y_{2})\)，
直线\(l\)的斜率为\(k\).\par
直线\(AB\)的方程为\(y = k(x - 1)\)，
代入\(\frac{x^{2}}{2} + y^{2} = 1\)，
得\((2k^{2} + 1)x^{2} - 4k^{2}x + 2k^{2} - 2 = 0\)，\par
其中\(\Delta = 16k^{4} - 4(2k^{2} + 1)(2k^{2} - 2) = 8k^{2} + 8 > 0\)，
\(x_{1} + x_{2} = \frac{4k^{2}}{2k^{2} + 1}\)，
\(x_{1}x_{2} = \frac{2k^{2} - 2}{2k^{2} + 1}\)，\par
\(y_{1} + y_{2} = k(x_{1} + x_{2}) - 2k = - \frac{2k}{2k^{2} + 1}\)，\par
从而\(C(\frac{4k^{2}}{2k^{2} + 1},\frac{- 2k}{2k^{2} + 1})\).因为\(C\)点在椭圆上，
代入椭圆方程得\(\frac{8k^{4}}{\left( 2k^{2} + 1 \right)^{2}} + \frac{4k^{2}}{\left( 2k^{2} + 1 \right)^{2}} = 1\)，\par
即\(k^{4} = \frac{1}{4}\)，
解得\(k^{2} = \frac{1}{2}\)，
所以\(x_{1} + x_{2} = 1\)，
\(x_{1}x_{2} = - \frac{1}{2}\).\par
所以\(|AB| = \sqrt{1 + k^{2}}\left| x_{1} - x_{2} \right| = \sqrt{1 + k^{2}} \cdot \sqrt{{(x_{1} + x_{2})}^{2} - 4x_{1}x_{2}} = \frac{3\sqrt{2}}{2}\)，\par
又点\(O\)到直线\(AB\)的距离为\(d = \frac{|k|}{\sqrt{1 + k^{2}}} = \frac{\sqrt{3}}{3}\)，\par
所以平行四边形\(OACB\)的面积\(S = |AB| \cdot d = \frac{3\sqrt{2}}{2} \times \frac{\sqrt{3}}{3} = \frac{\sqrt{6}}{2}\).\par
(ii)法一：由(i)可知\(C(1, - k)\)，
\(C\)关于\(O\)的对称点为\(D( - 1,k)\)，
\(A(x_{1},k(x_{1} - 1))\)，
\(B(x_{2},k(x_{2} - 1))\)，\par
因为\(x_{1} + x_{2} = 1\)，
\(x_{1}x_{2} = - \frac{1}{2}\)，\par
即\(A(x_{1}, - kx_{2})\)，
\(B(x_{2}, - kx_{1}) \cdot \overrightarrow{CA} = (x_{1} - 1,k - kx_{2}) = ( - x_{2},kx_{1})\)，
\(\overrightarrow{CB} = ( - x_{1},kx_{2})\)，\par
所以\(\overrightarrow{CA} \cdot \overrightarrow{CB} = (1 + k^{2})x_{1}x_{2} = - \frac{3}{4}\)，\par
\(\left| \overrightarrow{CA} \right| \cdot \left| \overrightarrow{CB} \right| = \sqrt{x_{2}^{2} + k^{2}x_{1}^{2}} \cdot \sqrt{x_{1}^{2} + k^{2}x_{2}^{2}} = \sqrt{(1 + k^{4})x_{1}^{2}x_{2}^{2} + k^{2}(x_{1}^{4} + x_{2}^{4})} = \sqrt{\frac{5}{16} + \frac{1}{2}(x_{1}^{4} + x_{2}^{4})}\)\par
又\(\overrightarrow{DA} = (x_{1} + 1, - k(x_{2} + 1))\)，
\(\overrightarrow{DB} = (x_{2} + 1, - k(x_{1} + 1))\)，\par
所以\(\overrightarrow{DA} \cdot \overrightarrow{DB} = (1 + k^{2})(x_{1} + 1)(x_{2} + 1) = \frac{9}{4}\)，\par
\(\left| \overrightarrow{DA} \right| \cdot \left| \overrightarrow{DB} \right| = \sqrt{{(x_{1} + 1)}^{2} + k^{2}{(x_{2} + 1)}^{2}} \cdot \sqrt{{(x_{2} + 1)}^{2} + k^{2}{(x_{1} + 1)}^{2}} = \sqrt{\frac{45}{16} + \frac{1}{2}({(x_{1} + 1)}^{4} + {(x_{2} + 1)}^{4})}\)，\par
根据\(x_{1} + x_{2} = 1\)，
\(x_{1}x_{2} = - \frac{1}{2}\)，
不妨设\(x_{1} = \frac{\sqrt{3} + 1}{2}\)，
\(x_{2} = \frac{1 - \sqrt{3}}{2}\)，\par
计算可得\({(1 + x_{1})}^{2} = 3x_{1}^{2}\)，
\({(1 + x_{1})}^{4} = 9x_{1}^{4}\)，
\({(1 + x_{2})}^{2} = 3x_{2}^{2}\)，
\({(1 + x_{2})}^{4} = 9x_{2}^{4}\)，\par
所以\(\left| \overrightarrow{DA} \right| \cdot \left| \overrightarrow{DB} \right| = 3\left| \overrightarrow{CA} \right| \cdot \left| \overrightarrow{CB} \right|\)，
所以\(\cos\langle\overrightarrow{CA},\overrightarrow{CB}\rangle = \frac{\overrightarrow{CA} \cdot \overrightarrow{CB}}{\left| \overrightarrow{CA} \right| \cdot \left| \overrightarrow{CB} \right|} = - \cos\langle\overrightarrow{DA},\overrightarrow{DB}\rangle\)，\par
所以\(\angle ACB + \angle ADB = \pi\).\par
根据对角互补的四边形为圆的内接四边形，可知\(A\)，\(B\)，\(C\)，
\(D\)四点共圆.\par
法二：曲线系解法\par
由(1)(2)得\(l_{AB}:y = k(x - 1)\)即\(kx - y - k = 0\)，
\(l_{CD}:y = - \frac{1}{2k}x\)即\(x + 2ky = 0\)，\par
所以过\(A\)，\(B\)，\(C\)，
\(D\)的二次曲线系为\(\frac{1}{2}x^{2} + y^{2} - 1 + \lambda(kx - y - k)(x + 2ky) = 0\)，
\(\lambda \in \mathbb{R}\)，\par
\(x^{2}\)，\(y^{2}\)，
*xy*的系数依次为\(\frac{1}{2} + \lambda k\)，
\(1 - 2\lambda k\)，\(\lambda(2k^{2} - 1)\)，\par
由(2)得\(k^{2} = \frac{1}{2}\)，
\(\therefore xy\)的系数\(\lambda(2k^{2} - 1) = 0\)，\par
又令\(\frac{1}{2} + \lambda k = 1 - 2\lambda k\)，
得\(\lambda k = \frac{1}{6}\)，
\(\therefore\lambda = \frac{1}{6k} = \pm \frac{\sqrt{2}}{6}\)，
\(\therefore\lambda\)存在.\par
\(\therefore A\)，\(B\)，\(C\)，\(D\)四点共圆.\par
注：曲线系设法不同，\(\lambda\)的取值不同，因此，
只需\(\lambda\)有解即可.}
\end{question}
