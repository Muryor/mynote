\end{array} \right.\)，即\(\{ \begin{array}{r}

\end{array} \right.\)，\par
解得\(- 1 < a \leq - \frac{1}{2}\)；\par

\end{array} \right.\)，即\(\{ \begin{array}{r}

1 < \frac{1}{a} \\
2 \geq \frac{1}{a}
\end{array} \right.\)，解得\(\frac{1}{2} \leq a < 1\)；\par
综上所述，\(a\)的取值范围为\(\left( - 1, - \frac{1}{2} \right\rbrack \cup \left\lbrack \frac{1}{2},1 \right)\).}
