# 一、单选题

1．已知$$\text{i}$$是虚数单位，则复数$$z = \frac{1 + \text{i}}{\text{i}}$$的实部为（　　）

> A．$$- 1$$ B．1 C．$$- \text{i}$$ D．i

【答案】B

【难度】0.94

【知识点】求复数的实部与虚部、复数的除法运算

【详解】复数$$z = \frac{(1 + \text{i}) \cdot ( - \text{i})}{\text{i} \cdot ( - \text{i})} = 1 - \text{i}$$，所以$$z$$的实部为1.

2．有一组样本数据1，2，2，2，3，5，去掉1和5后，相较于原数据不变的是（　　）

> A．平均数 B．极差 C．方差 D．中位数

【答案】D

【难度】0.94

【知识点】计算几个数的中位数、计算几个数的平均数
